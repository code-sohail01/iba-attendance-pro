# IBA CMS Attendance Pro

A premium, modern, and automated dashboard for Sukkur IBA University students to track attendance, calculate "bunk" capacity, and monitor academic standing in real-time.

## 🚀 Features
- **Automatic Background Sync:** Silent data fetching eliminates the need to manually refresh or select subjects.
- **Bunk Calculator:** Instantly shows your remaining safe skips for every course.
- **Master Dashboard:** View all your subject metrics in a clean, draggable, glassmorphism-inspired UI.
- **Smart Theme Engine:** Toggle between Light and Dark modes to match your workflow.
- **Stealth Mode:** Toggle the dashboard visibility with a single click on the extension icon.

## 🛠 Tech Stack
- **JavaScript (ES6+)**
- **Chrome Extension API (Manifest V3)**
- **DOM Manipulation & WebForms Scraping**
- **CSS3 (Glassmorphism & Flexbox)**

## 📦 Installation
1. Download the latest release as a ZIP file.
2. Unzip the folder.
3. Open Chrome and navigate to `chrome://extensions/`.
4. Enable **Developer mode** in the top right.
5. Click **Load unpacked** and select the folder you just unzipped.

## ⚖️ Disclaimer
This tool is for personal academic monitoring. Always verify your official attendance status through the official university portal.
