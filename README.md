# IBA Attendance Pro
A high-performance, context-aware Chrome Extension built for Sukkur IBA University students to track attendance and calculate safe skips in real-time.

## Features
* **Real-Time Sync:** Automatically scrapes and calculates attendance limits.
* **Smart UI:** Context-aware widget that only activates on the CMS portal.
* **Dark Mode:** Built-in theme toggling for the dashboard.

## Installation
1. Download the `IBA_Attendance_Pro_V1.zip` file.
2. Go to `chrome://extensions/` in your browser.
3. Enable "Developer Mode" in the top right.
4. Click "Load unpacked" and select the unzipped folder.
